Donkey Kong Country for Game Boy Advance is an exciting action game starring two of Nintendo's biggest stars: Donkey Kong and Diddy Kong.

Donkey Kong Country was originally released for the Super Nintendo Entertainment System in 1994, and it was immediately heralded as one of the best action games ever made. The game has been reproduced in its entirety for the Game Boy Advance, along with some extra bonuses not found in the original game.

Donkey Kong Country is primarily a single-player game, but there are some fun mini-games designed for two players with the use of a Game Boy Advance Game Link Cable.

Donkey Kong Country is available now.
